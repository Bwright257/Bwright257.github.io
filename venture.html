Venture

Venture is an open world role-playing game designed around simplicity. Starting in a mysterious forest, unaware of the world around you, you must discover the secrets of the land before death finds you.

Project Overview

Venture is an open-world adventure game developed using Java in Android Studio. Inspired by titles such as Rogue and The Ensign, Venture was created as a tribute to express my gratitude to the amazing developers of the aforementioned games. Starting originally as just a final project for a mobile app development class, I fell in love with the concept of the game and desire to hit the game's true potential. Exploration is the key aspect that Venture revolves around, and the vast open world hides many secrets.

Notable Features

Deceiving Simplicity

Obscured by two-bit graphics and a two-dimensional game world, a surprising amount of depth is hidden within the game's systems. Players, while being nothing but a @ symbol in the world of Venture, are able to level up and gear up to face off against increasingly difficult foes. Structures hidden around the world have tricks up their anthropomorphic sleeves and enemies seemingly have a will of their own, so players have to keep their eyes peeled for unexpected surprises!

Procedural Generation

Venture's open world is recreated procedurally every run of the game, leading to completely unique experiences that will statistically never be recreated. The game world is composed of a grid of so-called tiles. These tiles dynamically affect the procedural generation algorithm while it's running. After all tiles are generated, an entity spawning pass is done which populates the game world. This pass ties into the tile system as enemies and structures can only spawn on specific tiles unique to each entity. For example, the Snake enemy can only spawn in areas with large amounts of reeds along with having no nearby structures.
